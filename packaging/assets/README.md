# Packaging Assets

Optional application icons can be placed here before running PyInstaller.

Recognized names:

- `app.ico` or `icon.ico` on Windows
- `app.icns` or `icon.icns` on macOS
- `app.png` or `icon.png` on Linux

## Bundled UI font

`fonts/NotoSansSC-UI.ttf` is a UI-focused subset derived from Noto Sans SC. It is distributed under the SIL Open
Font License 1.1; the required license text is included at `fonts/OFL.txt` and is bundled by both PyInstaller build
modes.
